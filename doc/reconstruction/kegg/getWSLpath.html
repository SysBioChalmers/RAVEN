getWSLpath Translate a Windows-style path to its Unix WSL equivalent.

DESCRIPTION 

 getWSLpath  Translate a Windows-style path to its Unix WSL equivalent.

 Translate a Windows-style path to its Unix WSL (Windows Subsystem for
 Linux) equivalent.

 Parameters
 ----------
 path : char
     string with directory of file path, in Windows-style (e.g.

 Returns
 -------
 path : char

 Examples
